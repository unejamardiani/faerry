# Faerry Agent Workspace

This directory contains repo-local memory and coordination notes for agents
working on Faerry.

Use:
- `AGENTS.md` for active project instructions.
- `DESIGN.md` for the visual design system.
- `.agents/MEMORY.md` for durable decisions and context that should survive
  across future agent sessions.

Do not store secrets, tokens, customer data, or machine-local credentials here.
